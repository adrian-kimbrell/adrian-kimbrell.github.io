Savvy: Custom UI for Automatic 1111 Stable Diffusion API

Savvy is a custom HTML and JavaScript-based user interface designed for the Automatic 1111 Stable Diffusion API. This project was born from my desire to create a cleaner, more user-friendly alternative to the Gradio-based system, which I found too complex and tightly integrated into their ecosystem. Savvy focuses on simplicity, usability, and extensibility, providing a streamlined and intuitive experience for users. Key features include:
